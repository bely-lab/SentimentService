## SentimentService: Azure cloud Functions 

This project is a Python-based MS Azure Functions application that provides sentiment-related services through HTTP-triggered endpoints and a scheduled background task.

### Features
- AddSentiment: HTTP POST endpoint to submit sentiment data.
- GetResults: HTTP GET endpoint to retrieve sentiment results.
- DailySummary: Time-triggered function that runs based on a configurable schedule.

### Technologies Used
- Azure Functions (Python)
- Azure CLI
- Visual Studio Code
- Python 3.12
